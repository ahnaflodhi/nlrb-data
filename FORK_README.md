# NLRB Data Fork — LaborAudit Fallback Mirror

**DO NOT MODIFY THIS REPOSITORY**

This is a fork of [labordata/nlrb-data](https://github.com/labordata/nlrb-data) maintained
as a fallback data source for the [LaborAudit](https://laboraudit.com) platform.

## Purpose

The LaborAudit data pipeline (`scripts/refresh_nlrb_data.py`) downloads NLRB enforcement
data from this repo's nightly release. The download sequence is:

1. **Primary**: `labordata/nlrb-data` (upstream, maintained by Forest Gregg / Labor Data Initiative)
2. **Fallback**: `ahnaflodhi/nlrb-data` (this fork)
3. **Local**: Existing `nlrb.db` on disk (if both downloads fail)

## Sync Schedule

This fork should be synced with upstream periodically:

```bash
cd /home/ahnaf/projects/nlrb-data
git fetch upstream
git merge upstream/main --no-edit
git push origin main
```

## Critical Rules

1. **Never force-push** — preserve all upstream history
2. **Never modify data files** — the nightly release is auto-generated by upstream CI
3. **Never delete releases** — LaborAudit downloads from the `nightly` release tag
4. **Keep synced** — sync with upstream at least monthly to stay current
